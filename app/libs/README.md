فایل `libv2ray.aar` باید اینجا قرار بگیره.

- اگه از GitHub Actions استفاده می‌کنی (`.github/workflows/build.yml`)، این فایل
  خودکار در هر بیلد ساخته می‌شه و نیازی به کار دستی نیست.
- اگه محلی روی Android Studio بیلد می‌کنی، طبق مراحل README.md (بخش «قدم‌به‌قدم:
  وصل کردن هسته‌ی واقعی Xray») با gomobile بسازش و همین‌جا کپی کن.

این فایل عمداً در گیت commit نمی‌شه (نگاه کن به .gitignore) چون حجمش بالاست و
باید مطابق نسخه‌ی Xray مورد نظرت تازه بیلد بشه.
